## STM32duino-mini
mini version of stm32duino compatible maple mini

![stm32duino-mini](Images/STM32-Mini.jpg)


## Content

* Datasheet: chip documents
* Schematic: pdf file of board schematic
* Gerber: board gerber file
* Image: the board images
